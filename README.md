# Arnee personal website

Static personal website for Arnee.

## Deploy on GitHub Pages

1. Create a repository, for example `Paseditor.github.io`.
2. Upload `index.html`, `styles.css`, `script.js`, and the `assets` folder to the repository root.
3. Open repository settings and enable Pages from the `main` branch, root folder.
4. Update project links in `index.html` when you want to point each card to a specific repository.
